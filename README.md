
# Discord Logs

Em geral o bot captura todas as mensagens do servidor e loga em um site em tempo real.


## Instalacao/Start

```bash
git clone url
cd url
```

Frontend

```bash
  cd frontend
  npm i

  npm run start
```

Backend

```bash
  cd backend
  npm i

  npx ts-node ./src/server.ts
```

## Build

Backend
```bash
  cd backend
  npm i

  npx tsc
```

Frontend
```bash
  cd frontend
  npm i

  npm run build
```
